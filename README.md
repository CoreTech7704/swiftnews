## âš™ï¸ Environment Setup

# 1.Clone the repository

```bash
git clone https://github.com/CoreTech7704/swiftnews.git

```

# 2. Install dependencies
```bash
npm install
```

# 3. Add environment variables

Create a .env.local file in the root:


